О проекте Mathsem

Mathsem — это интерактивная образовательная платформа, разработанная студентами для изучения алгебры логики и теории множеств. Наш проект объединяет математическую теорию с практическим программированием, предлагая удобные инструменты для анализа логических выражений и работы с множествами.

Основные возможности:
✔️ Конструктор логических формул – визуальное построение выражений с операциями И, ИЛИ, НЕ.
✔️ Автоматическая проверка законов – например, законов де Моргана и других тождеств.
✔️ Примеры из теории множеств – операции над множествами (объединение, пересечение, дополнение).
✔️ Персонализированное обучение – сохранение прогресса в личном кабинете.

Технологии:

  • Frontend: HTML5, CSS3, JavaScript (адаптивный дизайн, SVG-графика).
  • Backend: Java (Spring Boot), MySQL (хранение данных пользователей).
  • Ядро логики: C++ (обработка сложных выражений).

Минимальные системные требования

  • ОС: Windows 7/8/10
  • Процессор: Intel Core i3 или аналогичный
  • ОЗУ: 4 ГБ
  • Место на диске: 500 МБ
  • Графика: OpenGL 3.3+

Для кого этот проект?

  • Студенты – помощь в освоении дискретной математики.
  • Преподаватели – наглядные примеры для занятий.
  • Разработчики – пример интеграции C++, Java и веб-технологий.

Проект разработан в рамках курса "Проектная деятельность" с акцентом на практическое применение математики в IT.
